HoGo REST APIs

Hogo Restful API makes it easy to integrate with your application to provide service for protecting, sharing and tracking digital documents(PDF, Word, Excel, and PowerPoint) via HoGo Cloud DRM services.
Please checkout API document in Github and get start for your development, now HoGo's providing you flowing API components:

  •             Authentication & register
  •             Upload/ Encode and store document
  •             Share documents & license setting
  •             Tracking document open
  •             Address book management
  •             Document management
